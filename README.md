# From Code to Software (Companion Repository)

This repository is the companion “digital lab” for the book **From Code to Software**.

The book teaches software development as a craft: ideas become programs, programs become systems, and systems survive by adapting to change. That means this repo is not a single “final answer.” It’s a history of deliberate iterations.

## How to use this repo
- **/src** holds the current evolving version of the project.
- **/iterations** holds snapshots that match the book’s chapter/iteration points.
- **/docs** is the companion website (GitHub Pages) with chapter references and links.
- **/exercises** contains optional practice.
- **/prompts** contains AI prompting patterns used in the book.

## Tags (important)
Each milestone is tagged so you can jump to the exact state of the code discussed in the book.

Tag format:
- `ch01-iter-01`
- `ch01-iter-02`
- `ch01-iter-03`
